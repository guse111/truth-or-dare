# Telegram Bot: Правда или Действие

🎮 Бот для игры в "Правду или действие" в Telegram

## Функции
- Регистрация игроков
- Генерация вопросов
- Подсчёт очков

## Технологии
- Python 3.10+
- aiogram 3.x
- SQLite

## Запуск
1. Clone репозиторий
2. Установи зависимости: `pip install -r requirements.txt`
3. Запусти: `python bot.py`
